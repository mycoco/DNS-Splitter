#### changelog
- 1.2.0.7
1. 启动程序后双击图标弹出窗体优化
2. 启动程序后自动增加防火墙规则
3. ip映射支持hosts格式,支持一个ip后面多个域名
```
127.0.0.1  www.test1.com  www.test2.com
```
4. 支持按端口分流: 优先尝试使用【域名:端口】进行匹配，未匹配到则只使用【域名】匹配
```
	192.168.0.100 www.test50.com:8000
	192.168.0.101 www.test50.com:9000
	192.168.0.102 www.test50.com
```
以及规则
```
	*test50.com:8000*  -> dns1
	*test50.com:8001*  -> dns2
```

5. 连接列表: 增加显示请求域名的端口以及匹配到的规则备注
